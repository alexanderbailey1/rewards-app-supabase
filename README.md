# Rewards App

This project is an Expo React Native application.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   # or
   npx expo start
   ```

3. Run the app on a device:
   ```bash
   npx expo run:android # Android device or emulator
   npx expo run:ios     # iOS device or simulator
   ```
